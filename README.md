# Clone repo
git clone https://github.com/planetgranito/invorce-edti.git

```bash
cd invorce edti

# Create virtual env
python -m venv venv
source venv/bin/activate  # or venv\Scripts\activate on Windows

# Install dependencies
pip install -r requirements.txt

# Run the app
start.cmd
```
